Javascript Exercise

Questions

  1. You have been given the task to manage a small library's book inventory. The library has a collection of books, and each book is represented as an object with the following properties: title (string): The title of the book. author (string): The author of the book. yearPublished (number): The year the book was published. available (boolean): Whether the book is currently available for borrowing. The library's book inventory is stored in an array named library.
    Create the library Array: Start by creating an empty array named library to store book objects. Add Books to the Library:
    Add the following book objects to the library array using the push method: "The Great Gatsby", "F. Scott Fitzgerald", 1925, true "To Kill a Mockingbird", "Harper Lee", 1960, true "1984", "George Orwell", 1949, false "Pride and Prejudice", "Jane Austen", 1813, true
  2. Update the Availability of a Book: You need to update the availability of "1984" to true by directly modifying the available property in the library array. Remove a Book from the Library: Remove "Pride and Prejudice" from the library array using the splice method.
  3. Check if a Book is in the Library:
    Check if "The Great Gatsby" is in the library using the includes method. Note that this method cannot be directly applied to objects, so think creatively about how to solve this.
  4. Borrow the First Book:
    Assume a user wants to borrow the first available book. Remove the first book from the library array using the shift method. Update its available status to false before removing it from the array, and then display the updated book details. Add a New Book to the Start of the Library: Add a new book "The Catcher in the Rye", "J.D. Salinger", 1951, true to the beginning of the library array using the unshift method.
  5. Get a List of All Book Titles: Extract a list of all the book titles from the library array and combine them into a single string, with each title separated by a comma, using the join method.
  6. Create a New Library Section: Create a new array called newArrivals containing only the last two books in the library array using the slice method.
